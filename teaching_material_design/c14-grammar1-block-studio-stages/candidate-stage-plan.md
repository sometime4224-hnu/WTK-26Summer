# C14 문법 1 보조 활동 표현 블록 맞추기 — 본 스테이지 후보안

상태: 교사 승인 완료 (2026-07-14)

승인 근거: 사용자가 후보안 전체를 승인하고, 스테이지 2의 2번을 `공해가 매우 많아요.`, 5번을 `친구들이 하도 기다려서 택시를 탔어요.`로 수정해 최종 확정했다.

## 보존 사항

- 현재 6문항 화면은 `튜토리얼 · 블록 모양 익히기`로 그대로 보존한다.
- 튜토리얼을 완료한 뒤 학습자가 `본 연습 시작`을 눌러 스테이지 1로 이동한다.
- 본 스테이지 1~3은 난이도와 문항 수가 같다. 각 스테이지는 `하도 … -아서/어서` 2문항, 단독 정도 부사 1문항, `-아서/어서` 1문항, `그래서` 2문항으로 구성한다.
- 자유 드래그, 정답 결합, 오답 튕김, 클릭·키보드 대체 경로를 모든 스테이지에서 유지한다.
- 스테이지 진행은 로컬 기기에만 저장한다.

## 본 스테이지 카드 규칙

- 미완료 카드 머리에는 번호만 보이고 안내 문장은 표시하지 않는다.
- 정답 결합 후 번호 오른쪽에 완성 문장을 크게 표시한다.
- 완성 문장 안의 배치 표현은 해당 블록 색과 굵은 글씨로 표시한다.
- 색만으로 정답을 구별하지 않도록 블록 모양, 점선/실선 윤곽, 결합 위치를 함께 유지한다.

## 스테이지 1 · 하루의 장면

1. 복도가 **하도 어두워서** 휴대전화 불을 켰어요.
2. 이 음식은 **진짜** 매워요.
3. 날씨가 추웠어요. **그래서** 두꺼운 옷을 입었어요.
4. 바람이 차가워서 창문을 닫았어요.
5. 영화가 **하도 무서워서** 텔레비전을 껐어요.
6. 버스를 놓쳤어요. **그래서** 택시를 탔어요.

## 스테이지 2 · 행동과 결과

1. 그 사람은 책을 **하도 많이 읽어서** 모르는 게 없어요.
2. 공해가 **매우** 많아요.
3. 라면이 뜨거웠어요. **그래서** 천천히 먹었어요.
4. 눈이 많이 와서 길이 미끄러웠어요.
5. 친구들이 **하도 기다려서** 택시를 탔어요.
6. 설명이 짧았어요. **그래서** 내용을 이해하지 못했어요.

## 스테이지 3 · 변화와 기억

1. 여기가 청계천이라고요? **하도 많이 변해서** 몰랐어요.
2. 그때의 기억이 **정말** 생생해요.
3. 어젯밤에 늦게까지 화면을 봤어요. **그래서** 눈이 빨개졌어요.
4. 옛날 사진을 봐서 추억이 생각났어요.
5. **하도 재미있어서** 시간 가는 줄 몰랐어요.
6. 친구의 모습이 많이 달라졌어요. **그래서** 처음에는 알아보지 못했어요.

## 화면 흐름

`튜토리얼 → 본 연습 시작 → 스테이지 1 → 스테이지 2 → 스테이지 3 → 전체 완료`

- 다음 스테이지는 앞 스테이지 완료 후 열린다.
- 마지막 정답 직후 강제로 이동하지 않고 완료 카드의 다음 버튼을 누른다.
- 이미 연 스테이지는 다시 방문할 수 있다.
- `되돌리기`와 `이 스테이지 초기화`는 현재 스테이지에만 적용한다.

## 반응형 기준

- 스마트폰 세로 320/390px: 단계 띠는 가로 스크롤, 블록 선반은 상단 고정 가로 도크, 문제 카드는 1열.
- 가로형 태블릿 1024×768: 블록 선반과 문장판을 나란히 배치하고 문제 카드는 1~2열 중 겹침이 없는 구성을 사용.
- 데스크톱 1440×900: 현재의 블록 선반 + 2열 자석판 구조를 유지.
- 200% 확대와 동작 줄이기 환경에서도 다음 행동과 완성 문장을 가리지 않는다.
