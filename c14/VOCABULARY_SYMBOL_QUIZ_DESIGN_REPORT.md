# 14과 상징 어휘 활동 제작 설계 보고서

## 1. 제작 범위

- 대상: `c14/vocabulary.html`에 있는 14과 핵심 어휘 28개 전체
- 학습 흐름: `상징 익히기` → `표현 맞히기` → `결과·재학습`
- 산출물:
  - `vocab-support-symbol-quiz.html`
  - `vocab-support-symbol-quiz.css`
  - `vocab-support-symbol-quiz.js`
  - `tests/e2e/c14-vocab-support-symbol-quiz.spec.js`
- 연결 위치: 14과 허브와 14과 어휘 카드 페이지의 상·하단 이동 영역

기존 농촌 산책 게임이나 도시·변화 어휘 연구소의 범위를 다시 나누지 않고, 이번 활동은 학습자가 28개 전체 어휘를 한 가지 시각 규칙으로 비교하고 회상하도록 설계했다.

## 2. 어휘 유형 분류와 활동 근거

| 학습 분류 | 어휘 수 | 분류 기준 | 상징 설계의 중심 |
|---|---:|---|---|
| 생활·분위기 | 10 | 시간, 기분, 생활 환경의 상태 | 시계, 표정, 시설, 거리, 공기의 대비 |
| 자연·농촌 대상 | 6 | 눈에 보이는 장소·식물·동물·일 | 대상 자체와 울타리·집·물 같은 배경 단서 |
| 농촌 동작 | 6 | 사람이 대상에 하는 행위 | 손·도구와 `→`, `⬇️`로 행동 방향 표시 |
| 존재·상태 변화 | 4 | 나타남, 없어짐, 달라짐, 재인 실패 | `먼저 → 나중`의 일관된 전후 변화 |
| 상상 가능성 | 2 | 머릿속 장면의 선명함 여부 | 생각 구름과 `✅`, `🚫`, `❓`의 대비 |

원본 어휘의 표기와 순서는 그대로 유지하되, 활동 안에서는 의미 처리 방식에 따라 다섯 묶음으로 재분류했다. 특히 `사라지다·생기다`를 농촌 동작과 분리하고 `변하다·몰라보다`와 함께 놓으면, 학습자가 변화의 방향과 결과를 비교하기 쉽다. `상상이 되다·상상이 안 되다`는 생각 구름의 선명도라는 별도 규칙으로 묶었다.

## 3. 28개 상징 체계

| 어휘 | 화면 표현 | 구별 단서 |
|---|---|---|
| 여유가 있다 | `🕘 ──── 🕔 ＋ 😌☕` | 두 시계 사이의 넓은 빈 시간 |
| 여유가 없다 | `🕘🕙🕚🕛 ＋ 🏃💦` | 빽빽한 일정과 급한 움직임 |
| 활기차다 | `🌞 ＋ 🙌🏃💃 ＋ ⚡` | 여러 사람의 힘찬 움직임 |
| 평화롭다 | `🏡🌿 ＋ 🕊️ ＋ 😌` | 시계 없이 고요하고 편안한 풍경 |
| 공기가 맑다 | `🏔️🌲 ＋ 🌬️✨ ＋ 😊` | 깨끗한 바람과 반짝임 |
| 공해가 심하다 | `🏭 ＋ 🚗 → 🌫️😷` | 오염의 원인과 마스크 |
| 따분하다 | `😐🪑 … 🕰️ … 🥱` | 멈춘 자세와 느린 시간 |
| 시간 가는 줄 모르다 | `🎨😊 ＋ 🕒 → 🕘❓` | 한 활동에 몰입한 동안 지난 시간 |
| 편의 시설이 잘되어 있다 | `🏠 ↔ 🏪🏥🚌 ＋ ✅` | 집 가까이 있는 여러 시설 |
| 불편하다 | `🏠 ──── 🏪🏥🚌 ＋ 😣` | 시설까지의 먼 거리 |
| 정원 | `🏠│🌳🌷🌿│🦋` | 집 옆 울타리 안의 꽃과 나무 |
| 잔디 | `🏠 ＋ 🟩 ＋ 🌱🌱🌱` | 땅을 촘촘히 덮는 낮은 풀 |
| 채소 | `🥬 ＋ 🥕 ＋ 🥒🍆` | 먹는 식물 자체 |
| 농사 | `🌱 → 🌾 → 🧺👩‍🌾` | 심기부터 수확까지의 전체 일 |
| 가축 | `🏠🌾 ＋ 🐄🐖🐐🐓` | 농장 안에서 기르는 여러 동물 |
| 물고기 | `🌊 ＋ 🐟🐟` | 잡는 도구 없이 물속 대상만 표시 |
| 가꾸다 | `🌱 ＋ 💧✂️👐 → 🌿🌷` | 물주기와 다듬기의 지속적 돌봄 |
| 깎다 | `🌿🌿🌿 ✂️ → 🌱🌱` | 긴 풀에서 짧은 풀로의 변화 |
| 심다 | `🖐️ ＋ 🌱 ⬇️ 🟫` | 흙 아래로 넣는 방향 |
| 키우다 | `👐 ＋ 🌱 → 🌿 → 🌳` | 돌봄 속에서 크기가 커지는 과정 |
| 짓다 | `👩‍🌾💪 ＋ 🚜↔ ＋ 🌾` | 농부가 논밭에서 직접 하는 동작 |
| 잡다 | `🎣 → 🐟 → ✋🪣` | 움직이는 대상을 붙드는 과정 |
| 사라지다 | `🔴 → ◯ → · → ∅` | 있던 대상이 마지막에 없음 |
| 생기다 | `∅ → · → ◯ → 🔴` | 없던 대상이 마지막에 나타남 |
| 변하다 | `🔵■ → 🟠▲` | 대상은 남고 모습·상태만 달라짐 |
| 몰라보다 | `🧒 → 🧔 ＋ 👀❓` | 달라진 사람을 알아보지 못한 결과 |
| 상상이 되다 | `🤔 → 💭🏡🌳☀️ ＋ ✅` | 또렷한 생각 구름 |
| 상상이 안 되다 | `🤔 → 💭🌫️❓ ＋ 🚫` | 흐리고 비어 있는 생각 구름 |

기호의 역할은 페이지 전체에서 통일했다.

- `→`: 시간·상태·행동 결과의 방향
- `∅`: 대상이 없음
- `────`: 먼 거리 또는 넓은 시간 간격
- `…`: 시간이 느리게 흐름
- `✅·🚫·❓`: 가능, 불가능, 불확실

## 4. 이미지 생성 여부와 성능 판단

사용자는 의미가 분명해지는 경우 생성 이미지를 쓸 수 있도록 요청했다. 검토 결과, 이번 활동에서는 다음 이유로 별도 래스터 이미지를 만들지 않았다.

1. 각 표현이 이모지와 방향 기호만으로도 핵심 의미를 전달한다.
2. 학습 목표가 사실적인 장면 관찰보다 `상징 ↔ 어휘`의 빠른 연결에 있다.
3. 단순 기호는 한 화면에서 전후 관계와 반대 방향을 더 일관되게 보여 준다.
4. 이미지 요청과 디코딩이 없어 28개 전체를 다뤄도 초기 화면이 즉시 열린다.

이미지만으로 애매할 수 있는 `농사·짓다`, `가꾸다·키우다`, `변하다·몰라보다`는 도구, 동작 주체, 결과 기호와 짧은 장면 단서를 함께 사용했다. 또한 `농사`와 `짓다`처럼 정지 장면에서 거의 같은 표현은 같은 문항의 직접 오답 쌍으로 두지 않았다. 이후 실제 수업 관찰에서 반복 혼동이 확인될 때에만 해당 항목을 512×384 WebP 장면으로 교체하는 것이 타당하다.

## 5. 학습 흐름 설계 근거

### 상징 익히기

- 한 번에 한 카드만 보여 주어 28개가 한꺼번에 주는 부담을 줄였다.
- 먼저 상징과 짧은 장면 단서를 보고, `어휘 보기`를 눌러 정확한 표제어와 쉬운 뜻을 확인한다.
- 마지막 카드의 다음 행동은 바로 `퀴즈 시작`이어서 재인 학습에서 회상 학습으로 자연스럽게 이어진다.
- 다섯 개 어휘 묶음 버튼으로 원하는 유형의 첫 카드에 바로 이동할 수 있다.

### 표현 맞히기

- 학습 때 본 같은 상징만 보여 주고 표제어와 뜻은 가린다.
- 보기 네 개는 무작위 관계없는 단어보다 같은 의미군·혼동군을 우선 사용한다.
- 오답 뒤에도 같은 문제에서 다시 선택하게 해 정답을 우연히 확인하고 끝내지 않게 했다.
- 점수는 최종 정답 수가 아니라 `첫 시도 정답`을 보여 주어 실제 회상 정도를 드러낸다.
- 완료 뒤 `헷갈린 어휘 보기`로 첫 시도 오답 카드에 돌아갈 수 있다.

## 6. 접근성·반응형 근거

- 이모지 각각은 화면 읽기 기능에서 숨기고, 상징 묶음 전체에 정답을 누설하지 않는 한국어 장면 설명을 제공한다.
- 모든 행동은 실제 `button` 요소이며 최소 44×44픽셀을 확보했다.
- 두 활동 탭은 좌우 방향키, 처음·끝 키로 이동할 수 있고 선택 상태를 `aria-selected`로 알린다.
- 정답·오답은 색만 쓰지 않고 `정답`, `다시 생각해 보세요` 문장으로 함께 알린다.
- 공용 라이브 영역에서 정오와 다음 행동을 짧게 안내한다.
- 320×568에서는 소개 문장을 숨기고 상징 높이를 줄여 첫 `어휘 보기` 버튼이 첫 화면 안에 나타난다.
- `prefers-reduced-motion` 환경에서는 애니메이션과 전환 시간을 사실상 제거한다.

## 7. 학습 기록 저장

고유 키 `c14-vocab-symbol-quiz-v1`에 다음을 자동 저장한다.

- 현재 학습 카드와 공개한 어휘
- 퀴즈 문제·보기 순서와 현재 위치
- 문제별 선택 기록, 정답 여부, 첫 시도 정답 여부
- 본 단서와 완료 상태

새로고침 뒤 같은 카드와 같은 문제 상태를 복원한다. 저장값에 알 수 없는 어휘나 범위를 벗어난 위치가 들어오면 유효한 28개 데이터만 남기고 안전한 값으로 정리한다. 전체 기록 삭제는 확인 창을 거쳐야 한다.

- 14과 허브와 어휘 페이지의 새 활동 연결
- 상징 공개와 학습 위치 자동 저장·복원
- 퀴즈에서 정답 어휘·뜻을 숨기는지 여부
- 오답 재시도, 선택 기록, 첫 시도 점수
- 28문항 완주, 결과 복원, 다시 풀기
- 오염된 저장값 정리와 기록 삭제 확인
- 활동 탭의 키보드 이동
- 320×568, 375×667, 768×1024, 1280×800에서 가로 넘침 방지와 첫 행동 노출
