# 조음 이미지 보강 에셋 제작 프롬프트

목표: `articulation-image-audit.md`에서 수정/보완 대상으로 분류한 항목을 앱 매핑 교체 전 단계에서 보강한다.

공통 스타일:

- 한국어 자음 조음 지도용 의학 교육 일러스트.
- 기존 앱의 측면 구강 단면도 스타일과 맞춘다.
- 흰 배경, 부드러운 반사 없는 색감, 선명한 혀/입천장/연구개/성문/입술 형태.
- 이미지 안에는 글자, 숫자, 라벨, 워터마크를 넣지 않는다.
- 조음 단서는 색상 하이라이트와 화살표만 사용한다.
- 파란색 화살표는 공기 흐름, 초록색은 접촉/접근 위치, 보라색은 입천장/연구개/성문 긴장 계열 단서로 사용한다.
- 4단계 시퀀스는 2x2 그리드로 생성한다. 각 칸은 같은 카메라 각도와 같은 해부학적 스케일을 유지한다.

## 1. 연구개/뒤혀 계열

### sequence-velar-soft-stop

ㄱ 예사 파열음 4단계. 2x2 그리드.

1. 준비: 혀가 낮고 구강이 열린 중립 상태.
2. 접근: 혀 뒤쪽만 연구개 쪽으로 올라가지만 아직 닿지 않음. 앞혀/치조 접촉 없음.
3. 폐쇄: 혀 뒤쪽이 연구개에 닿아 구강 뒤쪽을 막음. 연구개 접촉 부위만 초록색으로 강조.
4. 작은 방출: 폐쇄가 풀리며 짧고 약한 파란 공기 흐름이 입 밖으로 나감. 큰 기식 아님.

### sequence-velar-aspirated-stop

ㅋ 거센 파열음 4단계. 2x2 그리드.

1. 준비: 혀가 낮고 구강이 열린 중립 상태.
2. 접근/폐쇄: 혀 뒤쪽이 연구개에 강하게 닿음. 앞혀/치조 접촉 없음.
3. 압력: 뒤쪽 폐쇄 뒤에 공기 압력이 모임. 접촉 부위는 초록색.
4. 강한 기식 방출: 폐쇄가 풀리며 굵고 긴 파란 공기 흐름이 입 밖으로 크게 나감.

### sequence-velar-tense-stop

ㄲ 된 파열음 4단계. 2x2 그리드.

1. 긴장 준비: 혀 뒤쪽이 연구개 쪽으로 높게 올라가고 목/혀에 긴장감이 보임.
2. 단단한 폐쇄: 혀 뒤쪽과 연구개가 단단히 닿음. 접촉 부위가 진하게 강조됨.
3. 성문 긴장: 큰 공기 흐름 없이 성문/목의 긴장이 보라색으로 표시됨.
4. 짧은 무기식 방출: 매우 짧고 작은 파란 공기 흐름만 보임. ㅋ처럼 큰 숨이 아님.

### tongue-back-approach-clean

단일 이미지. 혀 뒤쪽이 연구개 쪽으로 올라가지만 아직 닿지 않은 장면. 앞혀와 치조에는 어떤 접촉 강조도 없음.

### place-velar-contact-clean

단일 이미지. 혀 뒤쪽과 연구개 접촉만 명확히 표시. 앞쪽 치아/치조 쪽에는 색 강조가 없어야 함.

### sequence-velar-nasal-coda

받침 ㅇ 4단계. 2x2 그리드.

1. 앞 모음 뒤 열린 상태.
2. 혀 뒤쪽이 연구개 쪽으로 접근.
3. 혀 뒤쪽이 연구개에 닿아 입길은 막히고, 연구개가 내려가 코길이 열림.
4. 닫힘은 유지한 채 코로만 울림/기류가 나감. 입 밖 파열은 없음.

## 2. 받침 불파 계열

### sequence-velar-coda-stop

받침 ㄱ 4단계. 2x2 그리드.

1. 앞 모음 뒤 열린 상태.
2. 혀 뒤쪽이 연구개로 접근.
3. 혀 뒤쪽이 연구개에 닿아 닫힘.
4. 닫힌 채로 끝남. 입 밖으로 나가는 파란 화살표 없음.

### sequence-front-coda-stop

받침 ㄷ 4단계. 2x2 그리드.

1. 앞 모음 뒤 열린 상태.
2. 혀끝이 치조 쪽으로 접근.
3. 혀끝이 치조에 닿아 닫힘.
4. 닫힌 채로 끝남. 입 밖으로 나가는 파란 화살표 없음.

### sequence-bilabial-coda-stop

받침 ㅂ 4단계. 2x2 그리드.

1. 앞 모음 뒤 입술이 열린 상태.
2. 두 입술이 서로 가까워짐.
3. 두 입술이 닫힘.
4. 닫힌 채로 끝남. 입 밖으로 나가는 파란 화살표 없음.

### airflow-no-release-stop

단일 이미지. 입안의 폐쇄 뒤 공기가 입 밖으로 나가지 않고 멈춰 있는 시각 단서. 파란 화살표는 입 안쪽에서 멈추며 입 밖으로 나가지 않는다.

### closure-held-final

단일 이미지. 닫힘을 유지하고 끝나는 상태를 보여주는 범용 카드. 외부 방출 화살표 없음.

## 3. 된소리 전용

### sequence-front-tense-fricative

ㅆ 된마찰음 4단계. 2x2 그리드.

1. 입술은 살짝 벌리고 혀 앞쪽이 치조 쪽으로 접근.
2. 혀 앞쪽과 치조 사이에 매우 좁고 팽팽한 틈 형성.
3. 성문/혀의 긴장 표시. 큰 방출 없음.
4. 얇고 강하게 조절된 마찰 흐름. 굵은 거센소리 화살표가 아니라 가늘고 집중된 파란 흐름.

### sequence-palatal-tense-affricate

ㅉ 된파찰음 4단계. 2x2 그리드.

1. 혀 앞쪽이 경구개 쪽으로 넓게 올라감.
2. 짧고 단단한 경구개 쪽 폐쇄. 치조 끝접촉만 보이면 안 됨.
3. 성문/혀 긴장 표시.
4. 큰 기식 없이 짧고 집중된 파찰 흐름.

### mouth-bilabial-tense-closure

단일 이미지. ㅃ용 두 입술의 강한 긴장 닫힘. 일반 입술 닫힘보다 입술 주변 압력/긴장이 보이되 과장된 표정은 피한다.

## 4. 초성 ㅇ / 예사 성문 보조

### zero-onset-vowel-start

단일 이미지. 초성 ㅇ처럼 아무 조음 위치도 막지 않고 바로 모음으로 시작하는 열린 구강. 접촉 강조 없음. 공기 흐름은 부드럽고 넓게 지나가며 특정 폐쇄 지점이 없어야 한다.

### zero-onset-open-vocal-tract

단일 이미지. 혀, 입술, 연구개 어디에도 폐쇄가 없는 열린 성도. 모음 시작 전 중립 상태.

### glottis-neutral-lenis

단일 이미지. 예사소리용 중립 성문. 성문이 과도하게 벌어지거나 강한 기식이 나오지 않는다. 약한 숨만 표시.

### glottis-soft-breath

단일 이미지. 예사 파열/파찰 보조용 약한 기류. 거센소리처럼 큰 파란 화살표가 아니라 짧고 희미한 흐름.
