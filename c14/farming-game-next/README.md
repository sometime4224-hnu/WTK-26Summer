# 소담한 마을 NEXT 프로토타입

기존 `c14/farming-game`을 바로 갈아엎지 않고, 그래픽과 게임 루프를 크게 키울 수 있는지 확인하는 별도 수직 프로토타입입니다.

## 목표

- 타일맵 기반처럼 보이는 넓은 월드
- 스프라이트 자산을 활용한 더 풍부한 마을 분위기
- 대화 → 채소 심기 → 물주기 → 성장 → 수확 → 저녁상 납품으로 이어지는 작고 완결된 루프
- 기존 어휘 학습 방향 유지: 표현을 퀴즈보다 행동 안에서 먼저 만나기

## 실행

정적 파일이므로 서버에서 다음 주소로 열 수 있습니다.

`/c14/farming-game-next/`

## 조작

- 이동: `WASD` 또는 방향키
- 행동: `Space` 또는 `E`
- 모바일: 하단 방향 버튼과 `행동` 버튼

## 의도적인 제한

- 기존 저장 데이터와 연결하지 않습니다.
- 기존 게임을 대체하지 않습니다.
- 엔진 전환 전 비교용이라 모든 기능을 완성하지 않았습니다.
