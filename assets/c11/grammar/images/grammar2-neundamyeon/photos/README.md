# 11과 문법2 설명 사진

이 폴더는 `c11/grammar2-explanation-photo-gallery.html`에서 쓰는 본문용 WebP 8장을 담습니다.

- 원본: 사용자 제공 PNG 8장, 1055 x 1491, 합계 약 12.96MB
- 본문용 출력: `photos/*.webp`, 1055 x 1491, 합계 약 1.22MB
- 썸네일 출력: `../thumbs/*-thumb.webp`, 320 x 452, 합계 약 0.19MB
- 변환 기준: ImageMagick WebP, 본문 quality 86, 썸네일 quality 78, metadata strip

원본 PNG는 live asset으로 복사하지 않았고, 페이지에서는 경량화된 WebP만 참조합니다.
